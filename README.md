## Project Title: Notion

A journal writing app

### Getting started

Fork the repository.

Clone it to your local machine.

Open a MongoDB server.

Under server folder, run _nodemon_.

Under client folder, run _npm i_ to install dependencies. Then run _npm start_.

Enjoy the app.
